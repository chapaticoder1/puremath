# DeepMath
This is a technical report of my path to learning the math behind ML models by building them from scratch with numpy. At the time of the beginning of this project, I am 18 years old, freshly passed out of the 12th grade, so the logs also contain derivations of concepts in multivariable calculus and statistics that were not covered in the math that I have studied so far.\
Project concept inspired by https://github.com/flackojodie/LogisticRegression-puremath

## 1. Logistic regression
Building a simple linear binary classification model for a blob dataset.\
Things covered and derived:
- Z-score Normalisation
- Binary Cross-Entropy Loss
- Sigmoid Activation
- Gradients of the Weights and Biases

## 2. Binary Classification with a Dense Neural Network
Building and testing different neural network architectures, activation functions, and initialisations for a titanic survival dataset.\
Things covered and derived:
- ReLU Activation
- He Initialisation
- Tanh Activation
- Xavier Initialisation
